# Currency-Converter
A modern, responsive currency converter web app that allows users to convert between different currencies using real-time exchange rates.

# Features
- Enter amount manually or via virtual keypad
- Convert between multiple currencies (USD, INR, EUR, GBP, JPY)
- Swap currencies instantly
- Real-time exchange rates using a public API
- Fully responsive (mobile-friendly UI)
- Clean and modern interface
- Error handling for API failures
- Loading state while fetching data

# How It Works
1. User enters an amount
2. Selects "From" and "To" currencies
3. App fetches live exchange rates from API
4. Displays converted result instantly

# API Used 
- https://api.exchangerate-api.com/v4/latest/{currency}
